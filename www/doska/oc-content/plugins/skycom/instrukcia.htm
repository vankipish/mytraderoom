Инструкция по установке скрипта SkyCom

1. Переписываем содержимое папки SkyCom в корневую папку на Вашем хостинге.

2. Вносим изменения в файл db.php

 

На примере локального Денвера расскажу где чего брать

1. Запускаем Денвер (или другой сервер)

2. В адресной строке набираем http://127.0.0.1/tools/phpmyadmin/ попадаем в панель управления базой данных MySQL (такая панель имеется почти на каждом хостинге)

3. Создаем базу данных введя ее название в окно:

вбиваем например newbaza и жмем создать.

 

4. Возвращаемся на главную страницу http://127.0.0.1/tools/phpmyadmin/ (щелкаем на домик слева)

и нажимаем ссылку Привелегии

 

далее нажимаем добавить нового пользователя

 

Заполняем следующим образом:

Имя пользователя: любое, например user

Хост: любой хост

Пароль: например 123

Обязательно нажимаем в привелегиях Отметить все и нажимаем Пошел

 

5. После создания пользователя и базы данных файл db.php заполняем следующим образом:

 

3. Заходим в панель управления базой данных MySQL (для Денвера: В адресной строке набираем http://127.0.0.1/tools/phpmyadmin/) и выбираем нашу базу.

Нажимаем ссылку SQL

 

Нажимаем обзор и находим файл SkyCom_db

Нажимаем пошел и все таблицы для работы скрипта будут созданы.

4.В файле index.php показан пример подключения комментариев. Для этого нужно:

  1. В самом начале страницы разместить следующий код:
    <? session_start(); include ("db.php"); ?>
  2. Между тегами <head> </head> подключить скрипты:
    <script type="text/javascript" src="scripts/jquery.js"></script>
    <script type="text/javascript" src="scripts/jquery.validationengine.js"></script>
    <script type="text/javascript" src="scripts/skycom.js"></script>
  3. Передать скрипту идентификатор, например ID Вашего материала
    <? $com='Ваш идентификатор'; ?> можно пользовать лат. буквы например nov_11
  4. И в нужное место Вашего документа вставить код публикации комментариев:
    <? include("skycom.php"); ?>

5.Обновление с версии 1.0 до версии 1.1:

  1. Заменить файлы: skycom.php, st.css (в нем есть пометка что изменилось),
    scripts/jquery.validationengine.js, scripts/skycom.js

Для входа в панель администрирования (SkyCom/adm/):

логин (e-mail для входа ): email@email.ru пароль: 12345

после входа поменяйте почтовый адрес (на него будут приходить уведомления о новых комментариях) и пароль.
